📖 Plain-English Glossary
Scenario
The whole automation from start to finish. Like a recipe — trigger + actions in order.
Module
Each individual step in a scenario. Could be a trigger (what starts it) or an action (what it does).
Operation
One time a module runs. Free plan = 1,000/month. A scenario with 3 modules running 100 times = 300 operations.
Bundle
One batch of data flowing through. If 5 form submissions come in at once, that's 5 bundles.
Mapping
Telling Make which piece of data goes where. Like copying a value from one field into another automatically.
Connection
A saved login to an app. You connect Google once, and Make uses it forever — you never re-login.
